\section{Forschung}
\begin{itemize}
\item Zunächst L2 Stabilität für nicht-lineare skalare Erhaltungsgleichungen bewiesen. 
\item Parallel an der Erweiterung für Systeme gearbeitet. Zunächst an linearen Systemen getestet + Wellengleichung
\item Anschließend erweitert auf Euler-Gleichungen -> Numerische Ergebnisse sehen gut aus. Allerdings noch keine theoretischen Resultate. Hier wäre entropie stabilität interessant
\item Ergebnisse zusammengetragen in Paper 
\item Damit befasst mehrere kleine Zellen hintereinander in 1D
\item Weitere Projekte: Limiter auf Cut Cells -> Forschung hier noch nicht sehr weiter. Es gibt keine Limiter höherer Ordnung auf Cut Cells. 
\item Weiterentwicklung der Methode in 2D. Lineare Advektion für höhere Polynomgrade scheint zu funktionieren -> Nur erste ergebnisse..
\end{itemize}
